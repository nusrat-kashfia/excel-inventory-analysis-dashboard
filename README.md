# Inventory Analysis Dashboard using Microsoft Excel

A portfolio project demonstrating practical **Microsoft Excel** skills for **Supply Chain**, **Inventory Management**, **Procurement**, and **Business Analytics** roles.

This project analyzes a large retail inventory dataset using **Microsoft Excel (Microsoft 365)** to generate business insights through **Pivot Tables**, **Pivot Charts**, **XLOOKUP**, **Slicers**, and an interactive dashboard.

---

# Project Overview

This project simulates the work of an **Inventory Analyst** or **Supply Chain Analyst** for a retail distribution company.

The dashboard helps management monitor:

- Product performance
- Inventory levels
- Sales performance
- Warehouse performance
- Purchase order analysis
- Supplier delivery performance
- Business KPIs

The project demonstrates how Excel can transform raw operational data into meaningful business insights that support data-driven decision making.

# Project Highlights

- Built an interactive Excel dashboard using Microsoft Excel 365

# Business Recommendations

Based on the dashboard analysis, the following recommendations were identified:

- Prioritize replenishment for products below reorder point.
- Reduce purchasing for slow-moving inventory.
- Monitor high-value inventory to reduce holding costs.
- Improve supplier performance by tracking delivery delays.
- Rebalance inventory across warehouses where appropriate.
- Use historical sales trends to improve demand forecasting.
